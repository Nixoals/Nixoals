#### Features
* Signin/login screen with email verification
* Market place of vintage clothes with multiples filters (price, era, brand, size)
* Locker with all the pieces you own, your favorites ones and personnal user information...
* Stripe payment and wallet account
* Full customizable backoffice (forest admin)
* MongoDB database
